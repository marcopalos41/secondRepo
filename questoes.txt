>>> O que é o MVC ?
https://en.wikipedia.org/wiki/Model%E2%80%93view%E2%80%93controller

É um padrão de arquitetura (architecture pattern). 
  model - manages the data, logic and rules of the application. receives user input from the controller
  view -  any output representation of information.  presents the model in a particular format.
  controller - accepts input and converts it to commands for the model or view.  receives the input, optionally validates it and then passes the input to the model. responds to the user input. 

Tem as vantagens de ter high cohesion (encapsulation) e low coupling (dependencies).
